# Pi Governance Adoption

This repository adopts AI-Native Engineering Operating System **2.2** from
`pcvantol/djconnect/docs/governance/PLATFORM_ARCHITECT_SYSTEM_INSTRUCTIONS.md`.
There are no local exceptions.

Definition of Done requires synchronized clean main, implementation-reality
check, applicable Python/pytest and service/display validation, release-bundle
and systemd startup evidence, updated rolling records/immutable history and
one reviewable PR.

Release profile: Linux/Pi deployment bundle or package, systemd startup and
display validation, GitHub/public Pi release distribution. Docker, Cloudflare
Workers and HACS are not Pi deployment targets.
