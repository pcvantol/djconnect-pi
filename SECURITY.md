# Security Policy

## Reporting a Vulnerability

Please report suspected security vulnerabilities privately by email:

security@djconnect.dev

Do not open a public issue for vulnerabilities, secrets, credentials, private
logs or other sensitive findings.

When possible, include:

- A short description of the issue.
- Steps to reproduce or verify the finding.
- Affected repository, version, release or commit.
- Potential impact.
- Any relevant logs or screenshots with secrets removed.

We will review reports as quickly as practical and coordinate fixes or
mitigations before public disclosure.

## Scope

This policy covers the DJConnect Raspberry Pi client code, installer, release
assets, local Client API, web portal and documentation in this repository.

For issues that affect the shared DJConnect product contract or Home Assistant
integration, please still report privately to security@djconnect.dev and note
the related repository or component if you know it.

## Supported Versions

Security fixes are targeted at the latest published DJConnect Pi release unless
otherwise stated by the maintainers.
